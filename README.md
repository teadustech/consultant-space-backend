# Live Consultant App (Starter Kit)

## Tech Stack
- Backend: Node.js with Express (server/)
- Database: MongoDB

## Getting Started
1. Rename `.env.example` to `.env` and update MongoDB URI
2. Run backend:
   ```bash
   cd server
   npm install
   npm start
   ```

Visit: backend API on http://localhost:5000
